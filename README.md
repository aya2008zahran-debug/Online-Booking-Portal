# ✙ Healthcare Appointment Booking Platform | منصة حجز مواعد طبية

A modern, professional, and user-friendly healthcare platform built with **React**, **Vite**, and **Vanilla CSS**. This project features a sleek Arabic-first (RTL) UI with smooth transitions and interactive booking flows.

## 🌟 Features | الميزات

- **Arabic RTL Support**: Fully optimized for Arabic reading experience using the **Cairo** Google Font.
- **Modern UI/UX**: Professional healthcare aesthetic with glassmorphism effects and soft shadows.
- **Doctor Search**: Intuitive hero section with search capabilities for doctors, specialties, and clinics.
- **Interactive Booking**: A 3-step simplified booking wizard (Specialty -> Doctor -> Date/Time).
- **Responsive Design**: Looks great on Desktop, Tablet, and Mobile devices.
- **No Dependencies Bloat**: Built with core React and lightweight icons (Lucide-React).

## 🚀 Getting Started | كيف تبدأ

### Prerequisites
- [Node.js](https://nodejs.org/) (v16.0.0 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation | التثبيت
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/healthcare-platform.git
   ```
2. Navigate to the project directory:
   ```bash
   cd healthcare-platform
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development | التطوير
Run the development server:
```bash
npm run dev
```

### Build | بناء المشروع للإنتاج
Create a production build:
```bash
npm run build
```

## 🛠 Tech Stack | التقنيات المستخدمة

- **Frontend Framework**: [React (Vite)](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Riyadh Design Standards)
- **Icons**: [Lucide-React](https://lucide.dev/)
- **Typography**: Cairo (Arabic), Inter (English)
